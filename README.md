
玩家操作指南
1. 進入遊戲 (CoverUI)
啟動後會看見「武器轉蛋遊戲」標題與閃爍的提示文字。
<br>操作：點擊畫面上任何地方即可進入登入頁面。

2. 帳號管理 (LoginUI / AddCustomerUI)
<br>註冊：初次遊玩請點擊「註冊」，系統會自動生成以 C 開頭的玩家 ID。
<br>登入：輸入帳號密碼。登入成功後，系統會將玩家物件序列化儲存於 Customer 檔案中。

3. 遊戲大廳 (GachaGameUI)
<br>玩家資訊：左上方會顯示玩家的動態 ID 與歡迎標語。
<br>即時時鐘：介面會顯示當前的系統日期與時間。
<br>功能導覽：可點擊「卡池」進入抽獎，或點擊「查詢紀錄」查看歷史戰績。

4. 卡池召喚 (GachaBannerUI)
<br>卡池切換：使用左右箭頭按鈕可切換不同的武器系列（如：亞洲父母、騎士王傳說）。
<br>新手卡池：
<br>規則：限定 10 連召喚，帳號限抽一次。
<br>保底：保證獲得一個常駐池的 SSR 道具。
<br>一般卡池：
<br>單抽：隨機獲得一件道具。
<br>10 連抽：若前 9 抽未獲得 SR 以上，第 10 抽將觸發 SR 以上保底機制。
<br>時間判定：若卡池不在資料庫設定的開放時間內，按鈕會自動變為灰色「尚未開放」。

GM 管理員後台 (GMUI)
<br>管理員帳號登入後可進入專屬後台，執行以下權限操作：
<br>管理員帳號必須自己從SQL裡面登錄，不是不小心沒做是故意沒做的。

1. 道具管理 (ItemCreatorView)
<br>新增道具：可輸入道具名稱、設定稀有度（SSR/SR/R/N）及中獎機率。
<br>系列分類：手動指定該道具所屬的卡池系列。

2. 卡池排期 (BannerManagerView)
<br>時間控制：管理員可針對特定卡池設定 start_time 與 end_time。
<br>目前"超絕武器神祭"是關閉的想吃自己打 可以從設定裡面開啟。
<br>即時生效：修改後，玩家端的卡池介面會立即同步顯示最新的開放時間。

3. 紀錄追蹤 (GMPlayerRecordQueryView)
<br>指定查詢：輸入玩家名稱即可調閱該玩家在所有卡池的詳細中獎清單。
